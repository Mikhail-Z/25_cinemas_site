API

API - это интерфейс, который позволяет получать информацию с сайта другим приложениям и сайтам. Синтаксис запросов и тип возвращаемых ими данных строго определены на стороне самого сайта, откуда предполагается брать информацию. На данном сайте тип возвращаемых данных - JSON.

Данный сайт позволяет получить список фильмов (10 штук) с необходимой информацией о них в формате json, обратившись по адресу.

Поля в этом объекте имеют следующее значение:
  • name - показывает назание фильма
  • cinemas_num - количесво кинотеатов Москвы,в которых оказывают эот фильм
  • url_on_afisha - ссылка на страницу с этим фильмом на сайте afisha.ru
  • score - средняя оценка фильма пользователями сайта afisha.ru по шкале от 1 до 10
  • genre - список жанров фильма (максимум 2 штуки)
  • image - ссылка на картинку, используемую в карточке фильма на сайте
